## Project Notes
Doctor Biz wants to split the claude-plugins monorepo (28 plugins) into separate GitHub repos. This is a major architectural change. Key considerations: marketplace.json currently references local paths, site generator reads from local dirs, meta plugins bundle other plugins, GitHub Actions auto-generates the site. The split affects discovery, installation, versioning, and the marketplace site generation pipeline. Need to understand the "why" before proposing approaches.
