## Task 1: GitHub Repo SEO — Topics, Description, License

**Files:**
- Create: `LICENSE` (MIT)
- Modify: GitHub repo settings via `gh` CLI

**Why:** The repo has zero topics, a generic description, and no license. This is free discoverability we're leaving on the table. Topics make the repo findable via GitHub search and improve the profile page appearance.

**Step 1: Add GitHub topics**

```bash

```

**Step 2: Verify topics appear**

```bash
gh api repos/2389-research/claude-plugins --jq '.topics'
```

Expected: Array with all 10 topics.

**Step 3: Update repo description**

```bash
gh repo edit 2389-research/claude-plugins --description "30 plugins and MCP servers for Claude Code — parallel exploration, iterative refinement, binary reverse engineering, structured decision-making, and more."
```

**Step 4: Update homepage URL**

```bash

```

**Step 5: Add MIT license**

Create `LICENSE` at repo root with standard MIT license text, copyright 2389 Research, Inc.

**Step 6: Commit license**

